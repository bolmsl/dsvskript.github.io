1.18. Slicing – Extrem hilfreich#

  • Slicing ermöglicht Zugriff auf viele Elemente

  • Slicing dezimiert die Elemente nach gewünschtem Muster

  • Syntax: start:end:step

  • Defaults

  • start: 0

  • end: letztes Element

  • step: 1

  • end wird nicht angenommen

  • bei negativem start oder end wird vom letzten Element aus gezählt

1.25.2. Beliebig lange Parameter-Listen#

  • häufig in Bibliotheken

  • Positionsparameter: *arg

  • Keyword Parameter: **kwargs

def varParam(*args, **kwargs):
    print('Positionsargumente:')
    for arg in args:
        print(arg)
    print('Keyword Argumente:')
    for kwarg in kwargs:
        print(kwarg, ':', kwargs[kwarg])
            

varParam('Test', 42, key='keyval', h=5)

Positionsargumente:
Test
42
Keyword Argumente:
key : keyval
h : 5
