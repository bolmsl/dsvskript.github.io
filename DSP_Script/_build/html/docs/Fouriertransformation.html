17. ▪ Fourier-Transformation ist eine Erweiterung der Fourier-Reihe für aperiodische Signale mit#

unendlicher Periodendauer 𝑇0 → ∞, 𝜔 → 0

▪ Das Spektrum besteht nicht mehr aus absoluten Amplitudenwerten an diskreten Frequenzen 𝑛𝜔0 (also Vielfachen der Grundfrequenz), sondern es besteht aus der Amplitudendichte über einer kontinuierlichen Frequenzachse.
